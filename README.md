# FPGA-Based Spherical Speaker Arrays

This repository contains the source files associated with the 2026 Linux Audio Conference paper:

Romain Michon, Maxime Popoff, Pierre Cochard, and Tanguy Risset. "Towards Affordable FPGA-Based Spherical Speaker Arrays." In *Proceedings of 2026 Linux Audio Conference*, Maynooth, Ireland, 2026.

It is organized as follows:

* `cad`: CAD (OpenScad and SolidWorks) files
* `faust`: Faust Ambisonics decoder
